# Data Analyst Assignment

## Overview
This project demonstrates my foundational skills in SQL, Google Sheets and Python by solving the real-world business problems for a Hotel and Clinic Management System.
This focuses on data analysis, problem-solving, and extracting meaningful insights.

----------------------------------------------------------------------------------------------------
## Skills Demonstrated

### SQL
-> Joins, Aggregrations, Filtering
-> Data Modeling, and schema design
-> Business queries and insights

### Google Sheets
-> VLOOKUP
-> Data and Time Functions
-> COUNTIFS
-> Data Analysis

### Python
-> Basic Scripting
-> Logic Building
-> String Manipulation

----------------------------------------------------------------------------------------------------

## Hotel Analysis
-> Last Booking Identification
-> Revenue Calculation per booking
-> High-value Bookings Detection
-> Item Usage Analysis
-> 2nd Highest Billing Analysis

----------------------------------------------------------------------------------------------------

## Clinic Analysis
-> Revenue by Sales Channel
-> Monthly revenue & Expense tracking
-> Profit Calculation
-> 2nd Highest Revenue Clinic
-> 2nd Least Profitable Clinic per State

----------------------------------------------------------------------------------------------------

## Spreadsheet Analysis
-> Data mapping using VLOOKUP
-> Same day and Smae hour ticket analysis
-> Outlet-wise Performance Tracking

----------------------------------------------------------------------------------------------------

## Python Scripts
-> Time Converter (minutes + hours)
-> Duplicate Character Remover

----------------------------------------------------------------------------------------------------

## Insights

### Hotel
-> Some bookings generate high revenue, indicating premium customers
-> Certain services are more frequently used

### Clinic
-> Online channel contributes higher revenue
-> Profitability varies across clinics and states

### Tickets
-> Many tickets are resolved within the same day
-> Few are resolved withing the same hour
-> Some outlets perform better in quick resolution
